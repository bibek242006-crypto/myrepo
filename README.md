# Portfolio Website

A simple personal portfolio website built with HTML, CSS, and JavaScript.

## Files

- `index.html` — main website page
- `styles.css` — layout and styling
- `script.js` — mobile menu and contact form behavior

## Usage

1. Open `index.html` in your browser.
2. Customize the name, content, and project details.
3. Optionally deploy to GitHub Pages or another static hosting provider.
